HTML5新增音频标记标记<audio>

<audio>语法格式如下：
<audio src="url" controls>替代文字</audio>
<audio> 于</audio>之间插入的内容是供不支持<audio>标记的浏览器显示的。

插入视频示例

<!DOCTYPE HTML>
<html>
<head>
<meta http-equiv="Content-Type" content="text/html; charset=utf-8">
<title>插入视频示例</title>
</head>
<body>
<h3>插入视频文件示例</h3>
<audio src="E:\Dreamweaver\file\music\艾辰 - 错位时空.mp3" style="width:400px; height:300px;" autoplay="autoplay" controls="controls"/>
此浏览器不支持audio标记
</audio>
</body>
</html>
