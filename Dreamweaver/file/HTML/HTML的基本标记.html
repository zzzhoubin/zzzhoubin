1. 标题文字标记：<hn>

    在HTML文档中,标题(Heading )很重要。标题是通过<h1>~<h6>共6对标记进行定义的。<h1>定义最大的标题，<h6> 定义最小的标题。

  2. 段落标记：<p>和换行标记：<br/>

    不论是在普通文档，还是在网页文字中，合理地划分段落会使文字显示更加美观，要表达的内容也更加清晰。在HTML文件中，使用段落标记<p>来描述段落。网页显示时，包含在<p></p>标记对中的内容会显示在-一个段落里。如果想另起一-行，可使用换行标记<br/>。

  3. 快标记：<div>和<span>

    <div>和<span>标记的作用都是用于定义页面内容的容器，用于编排页面布局，本身没有具体 的显示效果。这两个标记的显示效果由style属性或CSS来定义。

  4. 列表标记

    1. 有序列表标记<o1>

      有序列表是-一个项目的序列，各项目前标有数字以表 示顺序。有序列表由<ol></0l>标记对实
      现，在<ol></ol>标记之间使用成对的<li></i>标记添加列表项目。定义有序列表的语法格式如下。
      <ol type="" start="">
      <1i>列表信息</1i>
      <1i>列表信息</1i>
      <1i>列表信息</1i>
      ......
      </o1>
      默认情况下，有序列表的列表项目前显示1、2、....字号，从数字1开始计数。可以使用 type属性修改有序列表序号的样式，也可以定义start属性设置列表序号的起始值。

    2. 无序列表标记<ul>

      无序列表也是-一个项目的序列，不用数字而采用一个符号标志每个项目。无序列表由成对的<ul></ul>标记对实现，<ul></ul>标记之 间使用成对的<li></i>标记可添加列表项目。无序列表的语法格式如下。
      <ul type=">
      <1i>列表信息</1i>
      <1i>列表信息</1i>
      <1i>列表信息</1i>
      </u1>
      默认情况下，无序列表的每个列表项目前显示黑色实心圆点。可以使用type属性修改无序列表符号的样式，type 属性的具体取值及说明如表2-4所示，其中，type属性值必须小写。

    3. 自定义列表<dl>

      自定义列表不是一一个项目的序列，它是一系列项目和它们的解释。自定义列表以<dI>标记开始，自定义列表项目以<d>开始，自定义列表的解释以<dd>开始。自定义列表的语法格式如下。
      <dl>
      <dt>名称<dd>说明.
      <dt>名称<dd>说明
      <dt>名称<dd>说明
      .....
      </dl>
      <d>标记定义了组成列表项的名称部分，此标记只能在<dI>标记中使用。<dd>用于解释说明<dt>标记所定义的项目，此标记也只能在<dl>标记中使用。
