超链接类型

  1. 内部链接

    将超链接标记<a>中 href属性的URL值设置为相对路径，就可以在HTML文件中定义内部超链接。

  2. 外部链接

    外部链接指网页中的链接标题可以链接到网站外部的文件，需要定义外部链接时，在超链接标记<a>中,将其href属性的URL值设置为绝对路径即可。

  3. 书签链接

    如果有的网页内容特别多，页面特别长，需要不断翻页才能看到想要的内容，这时，可以在页面中(一般是页面的前部）定义一些书签链接。这里的书签相当于方便浏览者查看的目录，单击书签时，就会跳转到相应的内容。实际上，跳转的地址也可以是其他文档中的某一位置。
    在使用书签链接之前，首先要建立称为"锚记"的链接目标地址，格式如下：
    <a name="anchorname"></ a>
    在超级链接部分，指明用户定义的锚记名称，即可链接到指定的位置。

链接示例：

<!DOCTYPE HTML>
<html>
<head>
<meta http-equiv="Content-Type" content="text/html; charset=utf-8">
<title>链接示例</title>
</head>
<body>
<a href="http://www.baidu.com">百度</a>
<a href="http://www.luogu.com.cn">洛谷</a>
<a href="http://www.4399.com">4399</a>
</body>
</html>

  • 百度
  • 洛谷
  • 4399
