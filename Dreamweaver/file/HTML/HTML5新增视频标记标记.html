HTML5新增视频标记标记<video>

<video>语法格式如下：
 <video src="url" controls>替代文字</video>
如果浏览器不支持url指定的video元素，将显示替代文字。<video>标记常用的属性及说明：

插入视频示例

<!DOCTYPE HTML>
<html>
<head>
<meta http-equiv="Content-Type" content="text/html; charset=utf-8">
<title>插入视频示例</title>
</head>
<body>
<h3>插入视频文件示例</h3>
<video src="E:\Dreamweaver\file\video\大雨还在下.mp4" style="width:400px; height:300px;" autoplay="autoplay" controls="controls"/>
此浏览器不支持video标记
</body>
</html>
